# TG-Dashboard-Public
This is an alpha version of my development project, free to fork.

Installation instructions:                    
Copy the entire contents of the code on ```INSTALLATION.sh``` then type in the terminal like Termux, grant execution permission ```chmod +x setup.sh```.          

Create a file ```.env```, fill it with the following example:     

*$ nano .env*                          
```BOT_TOKEN=GET_IT_FROM_@BOTFATHER```     
```ADMIN_ID=YOUR_ID```
```METHOD_GROUP=https://t.me/...```
```OTP_GROUP=https://t.me/...```
```FLASK_SECRET_KEY=your_secret_key``` ```PORT=your_port```

Save it, ```ctrl+x+y``` enter, then run the command ```./setup.sh```.
